# Симулятор BIND9

Это веб-приложение, которое симулирует настройку DNS-сервера BIND9. 
![alt text](image.png)

Оно предоставляет удобный интерфейс для создания, редактирования и проверки конфигурационных файлов BIND9.
![alt text](image-1.png)
![alt text](image-2.png)

Приложение использует Google Gemini API для предоставления интеллектуальной обратной связи, включая проверку синтаксиса и объяснение конфигурационных файлов.

## Возможности

*   **Управление файлами:** Легко создавайте, редактируйте и удаляйте конфигурационные файлы BIND9 (например, `named.conf`, файлы зон).
*   **Проверка синтаксиса:** Получайте мгновенную обратную связь о вашей конфигурации BIND9. Приложение использует сервис на базе искусственного интеллекта для проверки синтаксических и логических ошибок.
*   **Объяснение конфигурации:** Поймите, что делает ваша конфигурация BIND9, с помощью четких и кратких объяснений, сгенерированных искусственным интеллектом.
*   **Многоязычная поддержка:** Пользовательский интерфейс и ответы, сгенерированные искусственным интеллектом, доступны на английском, испанском и русском языках.

## Технологический стек

*   **Фронтенд:** React, TypeScript, Vite
*   **Искусственный интеллект:** Google Gemini API
*   **Пользовательский интерфейс:** Пользовательские компоненты для редактора кода, вкладок файлов и консоли вывода.

## Начало работы

### Предварительные требования

*   Node.js, установленный на вашем компьютере.
*   Ключ API Google Gemini.

### Установка и настройка

1.  **Клонируйте репозиторий:**
    ```bash
    git clone <url-репозитория>
    cd bind9-simulator
    ```

2.  **Установите зависимости:**
    ```bash
    npm install
    ```

3.  **Настройте переменные окружения:**
    Создайте файл `.env.local` в корне проекта и добавьте свой ключ API Google Gemini:
    ```
    API_KEY=ваш-ключ-api-gemini
    ```

4.  **Запустите приложение:**
    ```bash
    npm run dev
    ```
    Приложение будет доступно по адресу `http://localhost:5173` (или другому порту, если 5173 занят).
